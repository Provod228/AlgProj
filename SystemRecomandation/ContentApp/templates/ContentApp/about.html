О нашем приложении

Content App — это интеллектуальная система рекомендаций товаров с использованием машинного обучения.

Наши возможности:

  • Персональные рекомендации на основе ваших предпочтений
  • Интеллектуальный поиск похожих товаров
  • Рейтинговая система с голосованием за категории
  • Избранные товары
  • Популярные товары по оценкам пользователей

Технологии:

  • Django + Django REST Framework
  • TensorFlow/Keras для нейронных сетей
  • scikit-learn для алгоритмов рекомендаций
  • Bootstrap 5 для интерфейса
